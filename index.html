Box-constrained static Google map

This is the getStaticGoogleMap function. It takes the following arguments:

  • coords - center coordinates for the map
  • sizingDiv - a div to use as base of map size
  • targetDiv - a div in which to put the map
  • linkto - an URL where the map will link when clicked
